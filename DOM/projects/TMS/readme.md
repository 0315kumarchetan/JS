# 📌 Task Management System (TMS)

## 🚀 Overview

Task Management System (TMS) is a simple web application to create, manage, and track tasks. It allows users to add tasks, categorize them, mark them as completed, edit, delete, and view task analytics.

---

## ✨ Features

* ➕ Add new tasks with category
* 📝 Edit existing tasks
* ❌ Delete tasks
* ✅ Mark tasks as completed
* 📊 View analytics (Total, Pending, Completed)
* 💾 Data persistence using LocalStorage
* 🎨 Theme toggle (Light/Dark mode)
* 📦 Responsive card-based UI

---

## 🛠️ Tech Stack

* HTML5
* CSS3
* JavaScript (Vanilla JS)
* LocalStorage (for data persistence)

---

## 📂 Project Structure

```
├── index.html
├── style.css
├── script.js
```

---

## ⚙️ How It Works

1. User adds a task using the form.
2. Task is stored in LocalStorage.
3. Tasks are rendered as cards.
4. Actions available on each card:

   * Edit
   * Delete
   * Mark Complete
5. Analytics update automatically based on task status.
